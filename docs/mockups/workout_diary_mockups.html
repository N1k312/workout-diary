Group 3 / Workout flow — core feature
Start · Active workout · Rest timer
The defining UX. Start = select exercises with drag reorder. Active = table of sets with custom numpad, auto-fill from last session, Prev column. Rest timer = bottom sheet with quick-set buttons, continues as banner when swiped down.
